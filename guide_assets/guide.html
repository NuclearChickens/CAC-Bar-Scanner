CAC Barcode Scanner — one-page quick reference

Made by Jeremy Evans

1Scan a CAC barcode

Aim the USB scanner at the Code 39 strip on the back of the Common Access Card. The app fires automatically when the scanner has sent all 18 characters.

Each scan decodes to EDIPI (10-digit DoD ID), category (PCC), and branch.

2Allowed scan

GREEN Banner flashes; this scan counts toward today’s drink total.

3Denied scan

RED Reason is shown under the banner; scan is not recorded.

4The seven tabs

ScannerThe verdict banner, input field, and decoded fields. Always returns focus to the input box. Hours Set bar open/close (HH:MM, 24-h) or switch to a rolling N-hour window. Equal open/close = 24-h day. Limits Max drinks per EDIPI per session (default 3). Once reached, further scans go red. Roster Tick the allowed categories (Active Duty, civilian, contractor, etc.) and branches. Use All / None for fast presets. Banned Per-EDIPI deny list. Blank Expires = permanent; YYYYMMDD for a date-bound ban. Reset Wipe every EDIPI’s drink count for the current window. Two-CAC authorized; logged forever. Logs Public audit trail: every unlock, settings change, lock, and reset — append-only.

5Unlock settings

LOCKED by default.
  1. Open any settings tab and click Unlock.
  2. Scan two different authorizing CACs.
  3. Banner turns UNLOCKED; edits auto-save.
  4. Click Lock when done — every change is logged.

6Reset drinks for the day

  1. Open the Reset tab.
  2. Click Reset drinks for the day.
  3. Scan two different CACs to authorize.
  4. Counts go to zero; the action is appended to the public log.
